Proyectos

Bee Happy.


Bee Happy es un proyecto que desarrollé en 2020, es un aplicación móvil y sitio web que ayuda a jóvenes adultos a conocer su nivel de depresión. El proyecto consta de una investigación que realice estudiando los factores más comunes de la depresión, realizar encuestas en más de 100 jóvenes y hacer ejercicios de minería de datos para desarrollar el algoritmo. Desarrolle la aplicación móvil con la herramienta de Android Studio, desarrolle el sitio web con HTML 5, CC3, JavaScript, Bootstrap y una conexión PHP. El desarrollo de interfaces fue realizado con Photoshop, Illustrator y Figma.

Epilepsiap.


EpilepsiaApp es una aplicación móvil que desarrolle para personas con problemas de epilepsia, para este proyecto realice un paper en inglés para el Congreso internacional de Ciencias Computacionales (CICOM 2018) y también publique en el Congreso nacionales para jóvenes investigadores (IMPACTA 2018). Realice ponencias presenciales en ambas publicaciones por lo cual tengo experiencia desenvolviéndome en entornos de inglés.
